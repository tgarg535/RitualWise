# RitualWise

RitualWise is a project built during the **Baidu Global Hackathon** using the **Medo** platform, where participants create products by giving natural-language instructions to an AI agent.

## 🌐 Live Project

**Project Link:** https://app-bi08uox0aigx.appmedo.com/

## 🧠 About the Project

RitualWise is designed to help users build and maintain meaningful routines/rituals with a simple, guided experience.

This project was created in a rapid hackathon environment by iterating through prompts and instructions to an agent on Medo, demonstrating how fast ideas can be turned into working products with agent-assisted development.

## 🏆 Hackathon Context

- **Event:** Baidu Global Hackathon  
- **Build Platform:** Medo  
- **Approach:** Instruction-driven development (prompting an agent to design and implement the app)

## ✨ Key Highlights

- Built quickly during a hackathon sprint
- Created using AI-agent instruction workflow
- Focused on practical routine and ritual support
- Deployed and accessible via public link

## 🚀 Tech & Workflow

Since this project was developed through Medo’s agent-based flow, implementation was driven primarily by:
- Natural language instructions
- Rapid iteration and testing
- Continuous refinement through prompt updates
